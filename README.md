## Project Description
This project is a simple **Pharmacy Management Website** created as part of an academic web development assignment.  
The website is designed using **HTML and CSS** and follows a consistent pharmacy-related theme across all pages.

The purpose of the website is to demonstrate:
- Proper website structure
- Navigation between pages
- Use of tables and forms
- Basic styling
- Collaboration using GitHub

## Website Pages
- **Home Page:** Introduction to the pharmacy, group member names, navigation menu, and images  
- **Table Page:** Displays pharmacy-related data such as medicine list or price table  
- **Form Page:** Includes customer details form with name, email, options, and submit button  
- **Additional Page:** About Us / Services / Contact page related to the pharmacy  

## Technologies Used
- HTML5  
- CSS 
- (Optional) Bootstrap used on one non-home page
